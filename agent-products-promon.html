行動安全管理

Promon SHIELD™ Mobile APP Security
保護行動應用程式

主動保護您的行動裝置免受目標性攻擊，能讓應用程式安全地運行，甚至在高度受感染的裝置上也是如此。當駭客攻擊您的裝置時，Promon SHIELD™ 將採取必要措施完全保護您的應用程式。

了解方案

需求痛點

行動 App 使用已成日常，使用者的個資、信用卡、憑證等機敏資訊，都會透過行動 App 來進行傳遞，其資料的保護是一個重要議題。此外，假冒、篡改的行動 App 氾濫，騙取、竊取客戶的資料，常常防不勝防。

效益與應用情境

行動 APP 加殼防護部署快、不影響開發流程，安全性高，支援靜態與動態防護。

隨著移動裝置爆發性的增長，使用者需要隨時隨地都能安全的使用行動服務，應用程式提供者需跟得上安全防護的挑戰，以避免遭受惡意軟體攻擊的風險。

Promon SHIELD™ Mobile APP Security 主要優勢

戰勝目標性攻擊

主動保護您的行動裝置免受目標性攻擊，能讓應用程式安全地運行，甚至在高度受感染的裝置上也是如此。當駭客攻擊您的裝置時，Promon SHIELD™ 將採取必要措施完全保護您的應用程式。

快速部署

Promon SHIELD™ 提供了一個自動化安裝啟用的程序。一旦整合後，Promon SHIELD™ 會篩選應用程式的營運邏輯、事件和資料串流，然後再將其綁定至現有代碼。這樣能讓公司組織快速發佈已受保護的應用程式，而不會影響開發時程表！

不影響使用者體驗

Promon SHIELD™ 保護多個商業行動裝置應用程式，並且不使用一種商用邏輯綁定單個行動裝置應用程式，它可以有效地擴展組織中的多個行動裝置應用程式，同時保持最佳的用戶體驗。

廣受全球一級客戶的信賴

Promon 適用於各種行業，擁有全球各行各業一級客戶，包括金融、醫療、物聯網和政府公部門等產業。Promon 獲得專利的深度保護技術 Promon SHIELD™，保護了超過 1 億用戶使用的行動裝置應用程式。

功能說明

  • RASP（Runtime application self-protection）
  • 惡意軟體
  • 偵錯器（Java Debugger, Native Debugger）
  • 模擬器 / 偽執行環境
  • 複製裝置
  • ROOT破解 / 越獄
  • 程式碼注入（預防運行時資料庫被注入惡意程式）
  • 編程-掛鈎
  • 重新包裝（偽造，操縱應用程式）
  • 系統與使用者營幕擷圖
  • 鍵盤記錄：不受信任的鍵盤
  • 鍵盤記錄和螢幕抹除：不受信任的螢幕閱讀器
  • Native Code-Hooks
  • 外部屏幕分享（內容顯示在裝置螢幕的「外部」，例如透過螢幕共享）
  • 資產完整性檢查：Promon SHIELD™ 可以對APK中的文件和資產執行
  • 深入的完整性檢查
  • Promon SHIELD™ 會在啟動應用程式時認證符合檔案的完整性
  • API（Application Programming Interface）：前景覆蓋偵測（層疊偵測）此功能可檢測是否有另
  • 一個應用程式放置在當前工作的應用程式前面，而讓其可執行網路釣魚攻擊。此種覆蓋攻擊， 眾所周知是由某些類型的Android惡意攻擊軟體製成的
  • 白箱加密功能，隨時用以防止「重要金鑰」顯現在記憶體中（可能被盜）
  • 從應用程式竊取敏感資料（處於靜止或其他時刻）
  • Man-in-the-App Scenarios
  • Man-in-the Middle Scenarion（中間人攻擊情境，與網路通訊有關）
