SRMAS
Smart Resource Monitoring and Alerting System

SRMAS is a system to monitor network traffic and manage all the data information from each device’s CPU and memory etc. Also, the important data will be seen in dashboards with alert notifications for uninterrupted operation.

Contact Us

Why does a company need SRMAS system?

For every IT technician, each server is located in different places with various functions. Hence, having a system to automatically monitor the operational status of all IT equipment and receive alerts is very important. The earlier we know what to repair, the less interrupted incidents will occur.

Before

  • Lack of Expertise: When network architecture becomes more complex, it will be harder to manage it cross-regionally.
  • Many branch offices Massive network traffic and devices need to be monitored, but there are only a few IT technicians…
  • Operational outages: It takes too long to repair the outages and results in high costs.

Key functions of SRMAS​

Device effectiveness management

  • Monitoring various server and network devices (Router, Switch, Firewall, Wireless AP).
  • Monitor host status through charts, including device temperature charts, fan speed charts, network connection diagrams, VLAN information, etc.

Alerts Widget

Customising alert notifications (Threshold). When the alert has exceeded the standard, the system will automatically send a notification to users via email or text message.

Quality control of network equipment

  • Monitoring the status of the internet and implementing the disconnection testing.
  • Sending the alert notification if needed.

Friendly Web UI to display map

  • Availability Map displays all data with coloured tiles, green up, yellow for warning, and red for down. You can also list all services and ignored/disabled devices in this widget.
  • Network-Weathermap builds network maps to help visualise network traffic flow rates.v

Server Stats Analysis

Collect CPU, MEMORY, NETWORK, and other information from various devices into SRMAS and use it for generating charts and queries. This data will be used for before-and-after comparisons to identify the causes of issues.
