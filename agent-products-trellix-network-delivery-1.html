網路傳遞安全

Trellix Network Security
入侵防禦系統

新一代的入侵預防系統 (IPS)，重新定義了組織封鎖進階威脅的方式。 不同於傳統的 IPS 解決方案，它超越了特徵碼比對的方式，而是利用多層式無特徵碼技術來抵禦未曾見過的威脅。 智慧型工作流程透過隔離威脅模式，讓安全管理員得以快速且正確地回應網路威脅與外洩，進而節省時間。

了解方案

海量的網路流量攻擊造成客戶網路中斷，並因此公司無法作業，尤其是因為目前流量都是加密的，一般的入侵防禦無法偵測。

Trellix：Network Security Platform 入侵防禦系統提供高效能處理器，解決加密流量無法偵測問題，並採用非 Signature 技術，阻擋未知攻擊

無可比擬的 IPS 保護與效能

使用網絡偵測和響應（NDR）不錯過任何攻擊

獲得無與倫比的可視性，應用最先進的無簽名檢測和保護技術，對抗包括零日攻擊在內的最高級和最具挑戰性的威脅。

回應重要警報

通過高度準確的警報提高分析師效率，在最需要時觸發警報，節省時間和資源，減少警報量和疲勞。

自動化和簡化安全工作流程

生成具體的即時證據和第七層元數據，提供更多的安全上下文，以進行調查和警報驗證、端點遏制和事件響應。

以最尖端的硬體平台為架構，可擴充到超過 320 Gbps 的速度，並採用 Security Connected 的方式，可利用來自多重產品的資料，因而能夠為最嚴苛的組織提供所需的入侵保護與可見度。

前所未見的入侵封鎖能力

透過特徵碼與無特徵碼檢查，阻止全新與未知的攻擊。 無特徵碼入侵偵測技術可讓 IPS 識別網路流量，進而阻止尚無特徵碼的前所未見攻擊。

更快速地偵測並修補外洩

減少調查安全性事件所需的專業知識與時間。 預先設定的引導式工作流程可協助 IT 人員更迅速地找到外洩所在，將警示引起的騷動降到最低，而專注於重新奪回網路控制權。

將安全與效能最大化

硬體平台是以最尖端的 Intel 技術為基礎，讓您擁有自訂安全性的自由，卻無需冒著輸送量損耗的風險。

填補安全性產品之間的缺口

Trellix Network Security Platform 是唯一提供 Security Connected 架構的 IPS，可整合多重產品。 在整個組織內共用安全資料，讓連線裝置可即時得知並回應威脅。
