雲端虛擬安全

Trellix Extended Detection and Response・XDR
擴展式偵測與回應

擴展式偵測與回應（Extended Detection and Response，XDR）是一個強大的資訊安全解決方案，可提供全方位的保護、偵測和回應能力。 XDR 的特色是將多個安全產品整合成一個統一的安全操作系統，讓企業能夠更簡單地查看整個技術風景線上的威脅情況，並提供即時可行動的威脅資訊，以加強安全操作的效能。

了解方案

資安危機 全面防堵

XDR 可以幫助 SOC 中心在遭受攻擊時改善重要的功能，包括：

偵測

結合端點遙測和多個安全控制提供商收集的安全事件，能夠識別更多有意義的威脅。

調查

利用人工智能以及機器協作，快速分析威脅信息，並幫助找出根本原因。

建議

提供分析師具體的調查建議，並給出相應的反應操作，以更有效地控制或消除檢測到的風險或威脅。

獵殺

通過跨數據庫存儲的通用查詢功能尋找可疑的威脅行為，幫助資安應變人員快速定位和處理風險。

常見企業痛點與需求

惡意軟體攻擊

駭客通過惡意軟體入侵企業系統，盜取機敏資料或對系統進行破壞

非法訪問

駭客利用漏洞或盜取密碼等方式進入企業系統，盜取資料或使其中斷營運

內部威脅

員工透過盜取機敏資料，或者不當使用系統等方式造成外洩風險

Trellix XDR 協助企業：

  • 自動化威脅檢測和識別，減少對人力資源的依賴。
  • 結合威脅分析和威脅情報，能夠更好地理解攻擊者的策略和技術，進而加強防禦措施。
  • 提供快速回應和解決方案，幫助企業在資安事件發生時迅速判斷與處置，減少損失。

Extended Detection and Response - XDR
提供先進的威脅偵測和反應功能，包括：

  • 偵測和回應針對性攻擊
  • 使用者行為和資訊資產分析
  • 威脅情報，包括本地共用威脅情報，以及外部獲取的威脅情報來源
  • 透過自動關聯性警示，減少追蹤假事件的時間
  • 整合相關數據以加速且準確地進行事件分類
  • 協作自動化以利簡化許多安全操作中心（SOC）流程
